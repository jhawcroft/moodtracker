Display and track changes in your mood.

== Description ==

Mood Tracker allows you to add a widget to your blog that displays your current mood.

<a href="http://wordpress.org/extend/plugins/contact-form-plugin/faq/" target="_blank">FAQ</a>
<a href="http://www.joshhawcroft.com/wordpress/plugins/mood-tracker/" target="_blank">Support</a>

= Features =

* Track changes in your mood as often as you like
* Five descriptive mood options to choose from: Terrible, A Little Low, Content, Good and Excited!
* View a line chart of your mood history for the past month/week
* Bright smiley icons show everyone who reads your blog how you feel

== Installation ==

1. Download the plugin if you haven't already.
2. Upload the plugin sources to your WordPress plugin directory (/wp-content/plugins/).
3. Activate the plugin through the Plugins menu in your WordPress administration console.
4. Drag the widget to a sidebar in the Widget menu of your administration console (Appearance &gt; Widgets).


== Frequently Asked Questions ==

= How can I configure the plug-in? =

There are currently no configuration settings for the plug-in.  It is a very simple widget.

= Why can't I see the plug-in after installation? =

Go through the <a href="http://www.wordpress.org/extend/plugins/jh-mood-tracker/installation/">Installation instructions</a> again and check you haven't made a mistake.

Make sure the sidebar that you installed the widget in is visible.

= How much mood data is kept? =

The plug-in keeps all the data you log, indefinitely.

= How much data does the graph show? =

The graph will show the last year worth of data, with averages for each week.  If only one week is available, it will show daily averages until more data is logged.
